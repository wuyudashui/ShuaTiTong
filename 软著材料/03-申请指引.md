# 刷题通智能刷题学习软件 V1.0 — 软著申请指引

## 一、申请流程

1. **注册账号** → 中国版权保护中心官网（https://www.ccopyright.com.cn）
2. **在线填报** → 选择"软件著作权登记" → "R11 计算机软件著作权登记申请"
3. **上传材料** → PDF 格式的源代码文档 + 用户操作手册
4. **提交审核** → 受理后 30 个工作日出证（加急可缩短到 10 个工作日）

---

| 字段 | 填写内容 |

## 三、源代码文档制作

已生成的 HTML 文档 `01-源代码文档.html` 包含全部源代码，共 **75 页**。

**转 PDF 步骤：**
1. 用 Chrome / Edge 打开 `01-源代码文档.html`
2. 按 `Ctrl + P`（或菜单 → 打印）
3. 目标打印机选择"另存为 PDF"
4. 更多设置：
   - 纸张大小：**A4**
   - 边距：**无**
   - 勾选"背景图形"
5. 保存为 PDF

## 五、常见被驳回原因

1. ❌ **页眉页脚不一致** — 每页页眉必须统一格式（页眉：软件名称 + 版本号）
2. ❌ **源代码有作者名/公司名** — 已清理干净
3. ❌ **截图和描述不匹配** — 截图后确认与文字说明对应
4. ❌ **文档内容不连贯** — 页码连续，不要缺页
